# AI Tutor 발표 자료

부대표급 임원 대상 프로젝트 소개 자료 묶음.

## 파일 구성

| 파일 | 용도 | 분량 |
|---|---|---|
| **AI_Tutor_프로젝트_소개.pptx** | 발표용 슬라이드 | 10장 + Thank you |
| **AI_Tutor_발표대본.md** | 슬라이드별 멘트 + 기본 예상 질문 | 약 12분 발표용 |
| **AI_Tutor_예상질문집.md** | Q&A 대비 카테고리별 55문항 | 발표 후 Q&A 용 |

## 발표 흐름 (10 슬라이드)

1. **Cover** — AI Tutor 정체성
2. **프로젝트 개요** — 왜 만들었나 + 솔루션 한 줄
3. **어떻게 만들어왔나** — 개발 사이클 + 마일스톤
4. **어떤 구조인가** — 시스템 아키텍처
5. **어떤 기술을 썼나** — 4 카테고리 스택
6. **어떤 모델을 썼나** — 8B 기본 / 32B 고급 / BGE-m3 / Edge-TTS
7. **어떻게 작동하나** — 사용자 흐름 + 내부 동작
8. **핵심 기능** — 분석 / 영상 / 챗봇 / 노트
9. **사용자 데이터 격리** — 4중 보안
10. **현재 상태** — 완료 9개 + 지표 5개

## 예상 질문 카테고리 (총 55문항)

- **A. 구현 (9)** — 어떻게 만들었나
- **B. 어려움 (8)** — 무엇이 까다로웠나
- **C. 기술 선택 (8)** — 왜 이걸 골랐나
- **D. 모델 & 성능 (6)** — 응답 속도 / 품질 / 한도
- **E. 데이터 & 학습 (5)** — 자료 형식 / 격리 / 누적
- **F. 보안 & 운영 (6)** — JWT / SQL / 유출 / 모니터링
- **G. UX & 사용성 (4)** — 조작 / 모바일 / 에러
- **H. 사업화 & 확장 (5)** — 시장 / 비용 / 차별화
- **I. 한계 & 다음 단계 (4)** — 기술 부채 / 로드맵

## 발표 직전 체크리스트

- [ ] 학교 서버 `bash run.sh` 떠있음 확인
- [ ] 사이드바에 `⚡ qwen3:8b (기본)` / `🔍 qwen3:32b (고급 탐색)` 둘 다 표시되는지
- [ ] cloudflared URL 외부 접속 가능 확인
- [ ] 시연용 강의자료 (1~3페이지 짧은 거) 준비
- [ ] 분석 미리 한 번 돌려둬서 결과 캐싱
- [ ] PPT 슬라이드 노트에 핵심 멘트 메모

## 시연 시퀀스 (3분)

1. **로그인** → 사이드바 드롭다운 보여주기 (10초)
2. **자료 업로드** → 자동 분석 + 영상 백그라운드 시작 토스트 (10초)
3. **요약 토큰 스트리밍** 라이브 시연 (30초)
4. **시험 탭** → 인터랙티브 카드 풀어보기 (30초)
5. **챗봇 탭** → 8B 로 질문 → 32B 로 같은 질문 → 답변 차이 보여주기 (60초)
6. **영상 탭** → 백그라운드 생성 완료된 영상 재생 (30초)

## 핵심 수치 (꼭 외울 것)

- 모델 4종 (LLM 2 + 임베딩 1 + Reranker 1) + TTS
- 자료 형식 **7종** (PDF/HWP/HWPX/PPTX/DOCX/TXT/MD)
- 학생 수준 **3단계** (입문/중급/심화)
- RAG 검색 **1초 이내** 응답
- 영상 **8분 길이 · 5분 생성** (8B 기준)
- 외부 LLM API 의존 **0개**

## 주의사항

- **모르는 질문은 솔직히** "다음 단계 과제입니다" — 추측 금지
- **숫자는 정확히** 외우기
- **부정적 질문**도 "리소스/시간 제약 + 우선순위 판단" 으로 차분히
- **마무리는 다음 단계로** — 진취적으로 보임
